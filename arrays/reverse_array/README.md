# Reverse array

input: Given an array (or string), the task is to reverse the array/string.

output: Return the reverse array.

**Example:**

Input  : arr[] = {1, 2, 3}
Output : arr[] = {3, 2, 1}

Input :  arr[] = {4, 5, 1, 2}
Output : arr[] = {2, 1, 5, 4}

Expected Time Complexity: O(|S|).
Expected Auxiliary Space: O(1).
